# Task Management System

A full-stack web application for managing tasks collaboratively in real time.

## Project Info
- **Course:** INTE 21323
- **Department:** Industrial Management

## Technologies Used
- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Database:** MySQL
- **Real-Time:** Socket.io
- **Authentication:** JWT + bcrypt
- **API Docs:** Swagger UI

## Setup Instructions

### Prerequisites
- Node.js v18+
- MySQL 8.0
- Git

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
node server.js
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
